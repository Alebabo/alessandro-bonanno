# Alessandro Bonanno

Personal portfolio site for Alessandro Bonanno, built as a static HTML/CSS page and deployed on Vercel.

## Contents

- Personal hero section with profile image and links
- AI-readable metadata via meta tags and JSON-LD
- Selected hackathon and AI product projects
- Responsive layout for desktop and mobile

## Deployment

The site is static. Vercel can deploy it directly from the repository root.
